# Visual Polish QA Skill

Bu skill, UI değişiklikleri sonrası görsel kalite kontrolü için kullanılır.

## Kontrol Edilecek Alanlar

- Typography hierarchy doğru mu?
- Başlıklar güçlü mü?
- Paragraph genişliği okunabilir mi?
- Section spacing yeterli mi?
- Kartlar hizalı mı?
- CTA görünür ve güçlü mü?
- Mobil görünüm bozuldu mu?
- Animasyonlar fazla mı?
- Hover efektleri tutarlı mı?
- Renkler premium görünüyor mu?
- Gereksiz kalabalık var mı?

## Premium UI Kontrolü

Şunlardan kaçın:
- Çok küçük yazılar
- Çok fazla border
- Rastgele gradientler
- Her kartta farklı stil
- Aşırı neon
- Çok hareketli arka plan
- Zayıf CTA
- Dengesiz boşluklar
- Mobilde sıkışan sectionlar

## Rapor Formatı

Her UI geliştirmesinden sonra şu formatta rapor ver:

1. Değiştirilen dosyalar
2. Eklenen görsel iyileştirmeler
3. Eklenen animasyonlar
4. Mobil kontrol notları
5. Performans riski
6. Daha sonra yapılabilecek polish önerileri
